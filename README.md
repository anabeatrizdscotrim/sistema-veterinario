# Projeto Sistema Veterinário

## Descrição
Este projeto é um sistema de cadastro de animais desenvolvido em Java para fins de estudos de testes unitários. Ele permite cadastrar, buscar, editar, excluir e listar animais em uma lista. Cada animal possui um nome, espécie, raça, idade, peso e ID único.

### Funcionalidades

- Cadastro de animais
- Busca de animais por ID
- Edição de animais
- Exclusão de animais
- Listagem de animais cadastrados

### Tecnologias Utilizadas
- Java
- JUnit (para testes unitários)

## Casos de Teste
O projeto inclui casos de teste para garantir o correto funcionamento das funcionalidades. Eles estão localizados no pacote test e podem ser executados usando uma estrutura de teste, como o JUnit.
